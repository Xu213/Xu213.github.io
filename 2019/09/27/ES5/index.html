严格模式

开头注明'use strict';

作用：消除了JS语法的不合理、不严谨之处，更加安全
语法和行为改变：

  • 必须用var声明变量，不能隐式声明
  • 禁止自定义函数中的 this 指向 window
  • 创建 eval() 函数作用域
  • 对象不能有重名属性

Object扩展

Object.create( prototype , [descriptors] )

作用：

  • 以指定对象为原型创建新的对象
  • 为新的对象指定新的属性，并对属性进行描述

eg：

Object.dcfineProperties( object , descriptors )

作用：

  • 为指定对象扩展多个属性
    get 取值 set 存值
    存取器属性 getter setter

Function扩展

Function.prototype.bind(<Object>)将函数内的this指向传入的<Object>(而不是window)，并将函数返回

区别call()与apply()和bind(): (相同点：都能改变this指向)

apply传入参数是数组对象，而call与bind是从第二个参数依次传入

bind不是立即调用，会将函数返回，call与apply会立即调用该函数
